# HotelManagement

Local hotel management WinForms application.

This repository contains 4 projects: `DAL`, `BUS`, `GUI`, and `DTO` targeting .NET Framework 4.7.2.

To run locally:
1. Open `GUI/GUI.sln` in Visual Studio 2019/2022.
2. Restore NuGet packages and build the solution.
3. Configure database connection in `GUI\App.config` or `DAL\App.config` if needed.

To push changes to GitHub, follow instructions in the root of the repository.
